* impl 뒤에 `{trait_name} for`를 추가하면 된다
    ```rust
    impl {trait} for {structure} {
      fn ...
    }
    ```

  * 기본 구현이라는 방법이 있다
    * 트레잇에서 함수의 원형뿐만 아니라 body까지 구현하고, impl시에 비어있는 중괄호를 사용하면 기본 구현을 자동으로 사용한다.
    ```rust
    pub trait Sample {
      fn sample() string {
        String::from("Hello")
      }
    }
    
    impl Sample for s {}
    ```
