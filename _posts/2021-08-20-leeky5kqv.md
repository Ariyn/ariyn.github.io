* Go에서 stdin을 한 줄씩 읽으려면 `bufio.Scanner`를 사용할 수 있다.
  * `os.Stdin`을 scanner에 넘기고, `Scan()`이 true인 동안 `Text()`를 읽는다.

```go
scanner := bufio.NewScanner(os.Stdin)
for scanner.Scan() {
    fmt.Println(scanner.Text())
}
if err := scanner.Err(); err != nil {
    log.Fatal(err)
}
```

* 줄 단위 처리에는 간단하고 읽기 좋다.
  * CLI 도구에서 pipe로 들어온 입력을 처리할 때 자주 쓴다.
* 매우 긴 줄을 읽어야 한다면 scanner buffer 제한도 확인해야 한다.
  * 일반적인 라인 입력에는 충분하지만, 큰 payload를 한 줄로 받는 경우에는 다른 reader가 더 맞을 수 있다.
