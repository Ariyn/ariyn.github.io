GOTO문은 하나의 함수 안에서만 사용이 가능하다에 대한 정리다.
핵심은 필요한 판단과 확인 지점을 짧은 명제로 남기는 것이다.

{: .tractatus}
1. 예시는 아래와 같다.

   ```go
     func main() {

     dest1:
         fmt.Println("Hello, playground")
         goto dest1

     // goto dest2 -> dest2 not defined error
     }

     func x() {
     // dest2: -> dest2 defined and not used error
     return
     }
   ```
