* 스타일 시트에 아래와 같은 코드를 넣고, markdown에서 `<p class="break-before"></p>`와 같이 사용한다.
  ```css
  .keep-together {
      page-break-inside: avoid;
  }
  
  .break-before {
      page-break-before: always;
  }
  
  .break-after {
      page-break-after: always;
  }
  ```
