MySQL CLI에는 SQL 뒤에 붙이거나 입력 중에 사용할 수 있는 편의 명령이 있다.
긴 결과를 터미널에서 볼 때나, 현재 입력 중인 쿼리를 확인할 때 유용하다.

{: .tractatus}
1. MySQL CLI에는 SQL 뒤에 붙이거나 입력 중에 사용할 수 있는 편의 명령이 있다.

   1. 긴 결과를 터미널에서 볼 때나, 현재 입력 중인 쿼리를 확인할 때 유용하다.

2. `\G`는 결과를 수직 형태로 출력한다.

   1. 일반 table 형태가 너무 넓어서 읽기 어려울 때 편하다.

   2. `SELECT * FROM users\G`처럼 쓴다.

3. `\p`는 현재 입력된 SQL을 다시 출력한다.

   1. 여러 줄로 쿼리를 작성하다가 지금까지 입력한 내용을 확인할 때 쓸 수 있다.

4. `\W`, `\w`는 warning 표시 여부를 바꾼다.

   1. warning을 바로 보고 싶으면 켜두고, 출력이 너무 지저분하면 끌 수 있다.

5. 이런 명령은 SQL이 아니라 MySQL client 명령이다.

   1. 다른 DB client나 driver에서는 그대로 동작하지 않을 수 있다.
