* `unknown time zone: Asia/Seoul`은 MySQL 서버의 타임존 설정 문제처럼 보이지만, 이 경우에는 Go 런타임이 `Asia/Seoul` 위치 정보를 찾지 못해서 발생한 오류였다.
  * DSN에 `loc=Asia%2FSeoul`처럼 지역명을 넘기면 `go-sql-driver/mysql`이 그 값을 `time.LoadLocation`으로 읽는다.
  * 컨테이너나 최소 설치 Ubuntu 이미지에 `tzdata`가 없으면 Go가 시스템 zoneinfo에서 `Asia/Seoul`을 찾지 못한다.
  * 그래서 호스트에서 Docker의 MySQL로 접속할 때는 되고, 같은 코드가 리눅스 환경에서는 실패하는 식으로 보일 수 있다.
* 드라이버 쪽 흐름은 단순하다.
  ```go
    // github.com/go-sql-driver/mysql@v1.5.0/dsn.go:451
    // connection string을 파싱해서 옵션들을 만들어내는 함수 parseDSNParams
    case "loc":
      if value, err = url.QueryUnescape(value); err != nil {
          return
      }
      cfg.Loc, err = time.LoadLocation(value)
      if err != nil {
          return
      }
    ```

  ```go
    // time/zoneinfo_read.go:529
    // parserDSNParams의 time.LoadLocation 함수가 마지막에 호출한다.
    // 이 함수는 Asia/Seoul 형식의 타임존 이름을 파싱하고, 시스템에서 값을 불러온다. 없다면 unknown time zone XXXX/XXXX 의 오류를 반환한다.
    func loadLocation(name string, sources []string) (z *Location, firstErr error) {
      for _, source := range sources {
        var zoneData, err = loadTzinfo(name, source)
        if err == nil {
          if z, err = LoadLocationFromTZData(name, zoneData); err == nil {
            return z, nil
          }
        }
        if firstErr == nil && err != syscall.ENOENT {
          firstErr = err
        }
      }
      if loadFromEmbeddedTZData != nil {
        zonedata, err := loadFromEmbeddedTZData(name)
        if err == nil {
          if z, err = LoadLocationFromTZData(name, []byte(zonedata)); err == nil {
            return z, nil
          }
        }
        if firstErr == nil && err != syscall.ENOENT {
          firstErr = err
        }
      }
      if firstErr != nil {
        return nil, firstErr
      }
      return nil, errors.New("unknown time zone " + name)
    }
  ```
* 해결 방법은 실행 환경에 타임존 데이터를 넣는 것이다.
  * Debian/Ubuntu 계열 컨테이너라면 `tzdata` 패키지를 설치한다.
  * Go 바이너리만 들고 가는 이미지라면 `import _ "time/tzdata"`로 zoneinfo를 바이너리에 포함시키는 방법도 있다.
  * 단순히 MySQL의 `time_zone` 값을 바꾸는 것만으로는 `time.LoadLocation("Asia/Seoul")` 실패를 해결하지 못한다.
