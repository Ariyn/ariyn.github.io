실행 중인 Docker 컨테이너 안으로 들어가려면 `docker exec`를 사용한다.
기본 형태는 `docker exec -it <container-id> /bin/bash`다.

{: .tractatus}
1. 실행 중인 Docker 컨테이너 안으로 들어가려면 `docker exec`를 사용한다.

   1. 기본 형태는 `docker exec -it <container-id> /bin/bash`다.

   2. 컨테이너에 `bash`가 없다면 `docker exec -it <container-id> /bin/sh`로 들어간다.

2. 먼저 `docker ps`로 실행 중인 컨테이너 id나 이름을 확인한다.

   1. 컨테이너 이름을 알고 있다면 id 대신 이름을 써도 된다.

3. 이 명령은 새 컨테이너를 만드는 것이 아니라, 이미 실행 중인 컨테이너 안에서 추가 프로세스를 실행하는 것이다.

   1. 디버깅할 때는 편하지만, 컨테이너 안에서 직접 고친 내용은 이미지에 자동으로 반영되지 않는다는 점을 기억해야 한다.
