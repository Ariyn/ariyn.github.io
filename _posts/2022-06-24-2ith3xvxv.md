예를 들어 `feature/foo`를 지우려면 `git push origin --delete feature/foo`다.

{: .tractatus}
1. 리모트 브랜치를 삭제할 때는 `git push origin --delete branch_name`을 사용한다.

   1. 예를 들어 `feature/foo`를 지우려면 `git push origin --delete feature/foo`다.

   2. 로컬 브랜치를 지우는 `git branch -d branch_name`과는 대상이 다르다.

2. 삭제 전에 브랜치가 이미 merge 되었는지 확인하는 것이 좋다.

   1. 공유 브랜치라면 다른 사람이 아직 사용 중인지도 확인해야 한다.

   2. 리모트 브랜치 삭제는 협업자에게 바로 영향을 준다.

3. 삭제 후에는 `git fetch --prune`으로 로컬에 남은 remote-tracking branch를 정리할 수 있다.
