* 공식 문서에서 나온 방법과는 차이가 있기 때문에, 공부할때만 주로 써야할 것 같다.
    * 이 방법으로 복잡한 코드를 실행하면 외부 라이브러리 같은 이슈가 있을 것
    * [출처](https://docs.scala-lang.org/overviews/scala-book/hello-world-2.html)
  ```javascript
  > vim main.scala
  object HelloYou {
    def main(args: Array[String]) = {
      println("Hello, World")
    }
  }
  
  > scalac main.scala
  > scala HelloYou
  "Hello, World"
  ```

    * 자바처럼 실제로 빌드된 파일은 오브젝트의 이름을 따라간다.
  * main args에 파라미터를 넣는 방법도 쉽다
    ```javascript
    > vim main.scala
    object HelloYou {
      def main(args: Array[String]) = {
        println(s"Hello, ${args(0)}")
      }
    }
    
    > scalac main.scala
    > scala HelloYou minuk
    "Hello, minuk"
    ```

  * 알고보니 간단하게 scala로 파일을 넘기기만 해도 실행된다...
    ```javascript
    > ls -al
      HelloYou.scala
    
    > scala HelloYou.scala
    ```
