* `Effective Java`의 저자는 조슈아 블로크다.
  * Java collection framework 설계에 관여한 인물로도 잘 알려져 있다.
  * 저자 소개 기준으로 썬과 구글에서 자바 엔지니어 및 시스템 설계자로 일한 이력이 있다.
* 이 책이 오래 읽히는 이유는 특정 문법 소개보다 API 설계와 객체 사용 습관을 다루기 때문이다.
  * equals/hashCode, 불변 객체, generic, enum, 예외 처리 같은 주제가 반복해서 나온다.
* Java 코드를 읽을 때 "왜 이렇게 짜는가"를 이해하는 데 도움이 되는 책이다.
  * 단순 레시피보다 설계 판단 기준을 배우는 쪽에 가깝다.
