1. 이경우 type이 error으로 바뀌기 때문에, type체크가 통과가 안된다

   1. https://github.com/pkg/errors/blob/v0.9.1/errors.go#L145

      ```go
        func WithStack(err error) error {
            if err == nil {
                return nil
            }
            return &withStack{
                err,
                callers(),
            }
        }

        type withStack struct {
            error
            *stack
        }
      ```

   2. 해당 부분의 코드를 보면 `&withStack{error}`을 리턴하는데, 이로 인해서 withStack타입 혹은 error타입으로 인식되버림

      ```go
        switch err.(type) {
          case ErrCustom:
            fmt.Println("custom")
          default:
              fmt.Println("default")
        }
      ```

   3. 이와같은 경우, default로 통과됨
