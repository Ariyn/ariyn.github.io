* Rust에서는 tuple을 사용해서 여러 값을 한 번에 반환할 수 있다.
  * 반환받는 쪽에서는 destructuring으로 각 값을 변수에 나눠 담는다.

```rust
let (a, b) = sample();

fn sample() -> (i32, i32) {
    (1, 2)
}
```

* 간단한 두세 개의 값은 tuple 반환이 읽기 쉽다.
  * 함수의 의미가 분명하고, 각 값의 역할을 호출부에서 바로 알 수 있을 때 잘 맞는다.
* 반환값이 많아지거나 의미가 복잡해지면 struct를 쓰는 편이 낫다.
  * tuple은 위치로 의미를 구분하므로, 값이 늘어날수록 실수하기 쉬워진다.
