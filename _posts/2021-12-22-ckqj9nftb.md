* Ubuntu에서 zlib 개발 헤더가 필요하면 `zlib1g-dev`를 설치한다.
  * C/C++ 확장 모듈을 빌드하거나, zlib에 링크되는 라이브러리를 컴파일할 때 필요하다.

```shell
sudo apt install zlib1g-dev
```

* 실행에 필요한 런타임 라이브러리만 필요하면 `zlib1g`가 대상이다.
  * 직접 빌드하지 않고 배포 패키지만 실행한다면 보통 런타임 패키지만으로 충분하다.
* `dev` 패키지는 헤더와 개발용 파일을 포함한다.
  * 빌드 오류에서 `zlib.h`를 찾지 못한다면 `zlib1g-dev`를 확인하면 된다.
