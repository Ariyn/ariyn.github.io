* json filter plugin을 사용하고 target을 지정하지 않으면 root로 들어가게 된다.
    ```conf
    filter {
      grok {
        match => "test %{GREEDYDATA:jsonString}"
      }
      json {
        source => "jsonString"
      }
    }
    ```

  * https://www.elastic.co/guide/en/logstash/current/plugins-filters-json.html#plugins-filters-json-target
