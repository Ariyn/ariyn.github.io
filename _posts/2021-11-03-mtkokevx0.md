json filter plugin을 사용하고 target을 지정하지 않으면 root로 들어가게 된다.
핵심은 필요한 판단과 확인 지점을 짧은 명제로 남기는 것이다.

{: .tractatus}
1. json filter plugin을 사용하고 target을 지정하지 않으면 root로 들어가게 된다.

   ```conf
       filter {
         grok {
           match => "test %{GREEDYDATA:jsonString}"
         }
         json {
           source => "jsonString"
         }
       }
   ```

   1. https://www.elastic.co/guide/en/logstash/current/plugins-filters-json.html#plugins-filters-json-target
