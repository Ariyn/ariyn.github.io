* GoLand에서 `gofmt`가 저장 시 동작하지 않으면 File Watcher 설정을 먼저 확인한다.
  * GoLand 자체 formatting 설정과 File Watcher 설정이 별도로 꼬여 있을 수 있다.
  * 외부 `gofmt`를 실행하도록 해두었다면 실행 파일 경로와 인자가 정확해야 한다.
* File Watcher의 `gofmt` 설정은 아래 값들을 확인한다.
  * program: `/path/to/go/gofmt`
  * arguments: `-w $FilePath$`
  * output path to refresh: `$FilePath$`
* `-w`가 빠지면 결과가 파일에 저장되지 않고 stdout으로만 나올 수 있다.
  * formatter가 실행된 것처럼 보여도 실제 파일이 바뀌지 않는다면 이 부분을 의심한다.
* 설정을 바꾼 뒤에는 일부러 formatting이 필요한 Go 파일을 저장해서 실제로 바뀌는지 확인한다.
