* 따라서 변경할 수 있는 변수를 선언하기 위해서는 mut를 붙여줘야 한다
  * 심지어 포인터 `&`도 불변이다
    * 변경할 수 있는 포인터 변수는 `&mut a`가 되어야 한다
  ```rust
  let a = 1 // 불변
  let mut b = 2 //변경 가능
  
  &a // 불변
  &mut b // 변경 가능
  
  &mut a // 사용 불가. 불변으로 선언된 변수를 변경가능하게 바꿀 수 없음
  ```

    * 흐음... 만에하나 불변으로 선언한 변수를 변경해야하는 일이 생기면 어떻게 될까?
