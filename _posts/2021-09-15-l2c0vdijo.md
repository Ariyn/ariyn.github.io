* -N옵션
    ```javascript
    mysql -N -e 'SELECT * FROM user;'
    ```

  * -column-names 
    * Write column names in results.
    * (Defaults to on; use --skip-column-names to disable.)
  * N, --skip-column-names
    * Don't write column names in results.
  * 실수로 -n 옵션을 주게되면 다른게 적용되니 조심하자
    *  -n, --unbuffered
      * Flush buffer after each query.
