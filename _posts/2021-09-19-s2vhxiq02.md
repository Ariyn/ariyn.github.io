* 간단한 static web server가 필요할 때는 Python 내장 모듈을 사용할 수 있다.
  * 파일을 제공하고 싶은 디렉터리로 이동한 뒤 실행하면 된다.

```shell
python3 -m http.server 8000
```

* 예전 메모처럼 80번 포트를 쓸 수도 있지만, 보통은 관리자 권한이 필요하다.
  * 개발 확인용이면 `8000`, `8080` 같은 포트를 쓰는 편이 편하다.
* 이 서버는 정적 파일 확인용이다.
  * 인증, 캐시 제어, HTTPS, 라우팅이 필요한 운영 서버 용도로 쓰면 안 된다.
