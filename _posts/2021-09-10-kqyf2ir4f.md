* `jq`에서 여러 필드를 한 번에 뽑고 싶다면 string interpolation을 사용할 수 있다.
  * 예시는 `jq '.lists[] | "\(.first) \(.last)"'`다.
  * 각 객체에서 `first`와 `last` 값을 꺼내 하나의 문자열로 만든다.
* 단순히 JSON 객체 형태로 유지하고 싶다면 `{first, last}`처럼 새 객체를 만드는 방법도 있다.
  * 사람이 읽을 출력이면 문자열이 편하고, 다시 프로그램에서 처리할 값이면 JSON 형태가 낫다.
* shell pipeline에서 다음 명령으로 넘길 값이라면 출력 형식을 먼저 정해야 한다.
  * 공백이 들어갈 수 있는 값은 문자열 조합보다 `@tsv`나 JSON 배열이 더 안전할 때가 있다.
