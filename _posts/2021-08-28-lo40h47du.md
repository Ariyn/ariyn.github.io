```shell
  > awk '1' < /tmp/test_file
  test
  > cat /tmp/test_file | awk '1'
  test
  ```

  * 파일의 컨텐츠를 읽어서 stdin으로 넣는것
    * 조금 다른 경우가 있지만, `cat file | command`과 동일한 역할로 보임
