# ABOUT

황민욱의 블로그입니다.

저는 백엔드 개발자로 일하며, 도구·자동화·AI·지식관리·제품 아이디어에 관심이 있습니다.
이곳에는 개발 중 배운 것, 개인 도구를 만들며 얻은 기록, 책과 개념에서 출발한 사유를 남깁니다.

완성된 글보다는 생각의 흔적에 가깝습니다.
Dev는 기술적 문제 해결 기록이고, Personal은 개념과 삶의 문제를 구조화해보는 공간입니다.

## 저는 이런 것을 기록합니다

1. 개발 중 만난 실제 문제
2. 개인 도구와 자동화 실험
3. AI 시대의 개발자 역할
4. 철학·인지·커리어에 대한 사유
